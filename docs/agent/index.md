# 什么是智能体？

智能体（Agent）就像一个会思考和行动的虚拟小助手。它能感知周围环境的信息（比如你输入的问题、上传的文件、网页内容等），然后根据预设的大脑（规则、模型或知识库）自己决定怎么做（比如回答问题、生成内容、处理数据），最后把结果输出（输出答案或执行任务）。
